Добро пожаловать!

~~~
Правила игры
- Игровое поле состоит из сетки размером 7х7.
- Главная цель взорвать все корабли соперника первым.
- Перед началом игры, участники размещают свои корабли на поле.
- Корабли разрешено ставить минимум на расстоянии 1 клетки друг от друга.
- Участники игры ходят поочередно, пропуск хода кем-либо недопустим.
- В свой ход участники стреляют по свободным ячейкам.

Это домашнее задание для школы skillfactory.
Я чутка нарушил правилам задания и сделал поле игры 7x7, чтобы комфортнее было расставлять корабли.

Приятной игры!
